Welcome to the unhosted SDK.

This page will help you install the required configuration on your webserver. These instructions assume you are using Apache2 on a Debian server.

Install PHP

  • sudo apt-get install libapache2-mod-php5

Install .well-known/host-meta

  • to install webfinger: sudo ln -s /var/www/unhosted-subdir/.well-known /var/www/.well-known

Put CORS headers on .well-known/host-meta

  • to install CORS: sudo a2enmod headers
  • to activate the .htaccess files, configure something like this apache config

Put DAV on /webdav

  • to install DAV: sudo a2enmod dav_fs
  • to activate DAV on the /webdav directory, configure something like this apache config

Activate https

  • to install CORS: sudo a2enmod ssl
  • to activate the https virtual host, configure something like this apache config
  • register an SSL cert, these used to cost money, but can now be obtained for free at www.startssl.com
  • upload the three files (cert, key, and chain)

Restart Apache and retry

  • sudo apache2ctl restart
  • refresh this page
